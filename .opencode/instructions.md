# Poneglyph Project Rules — Caveman Ultra

Override global AGENTS.caveman.md default.

Caveman ultra mode active by default in this project. Use `/caveman lite|full|ultra` to switch.

Ultra rules: abbreviate (DB/auth/config/req/res/fn/impl), strip conjunctions, arrows for causality (X → Y), one word when one word enough.

To disable: "stop caveman" or "normal mode".
